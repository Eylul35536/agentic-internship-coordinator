# Agentic Internship Coordinator System

AI-powered internship coordination platform designed to automate candidate evaluation, internship matching, and communication workflows.

## Planned Tech Stack

* Frontend: React
* Backend: FastAPI
* AI Framework: LangGraph
* Workflow Automation: n8n
* Email Integration: Gmail
* Database: TBD (SQLite / PostgreSQL)

## Planned Agents

* Coordinator Agent
* CV Screening Agent
* Candidate Evaluation Agent
* Matching Agent
* Report Generation Agent
* Email Decision Agent

## High-Level Workflow

Candidate → Frontend → FastAPI → LangGraph Agents → n8n → Gmail
